Application: Movies

The Movies app does the following:

  1. The app imports movie data from file: "importedmovies.json" ( subset already downloaded from link: https://raw.githubusercontent.com/prust/wikipedia-movie-data/master/movies.json), when started for first time.
  2. User can view and add movies including Casts and Genre
  3. The app uses classes, instances and variables. 
  4. The app allows for manipulation and persisting of data in files.
  
To run the app, please execute using the file: program.cs and follow the menu prompts.
